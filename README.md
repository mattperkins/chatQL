### Root Query
```

  users {
    id
    name
    email
    messages {

    }
  }
}
```

### getUser Query
```
query getUser($id: ID!){
  user(id: $id) {
    email
    name
  }
}
# QUERY VARIABLES (graphiql bottom GUI tray):
{ "id": "1" }
```
